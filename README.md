## Clone project

Clonar el proyecto:

```
git clone https://github.com/telematico87/prueba-_progrez
```

Luego:

```
cd prueba-_progrez
npm install
npm start
```

You should see the app in your browser address bar at [http://localhost:3000](http://localhost:3000)
