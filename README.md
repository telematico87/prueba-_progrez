## Instalación

Clonar el proyecto:

```
git clone https://github.com/telematico87/prueba-_progrez
```

Luego:

```
cd prueba-_progrez
npm install
npm start
```

Seguidamente puede acceder a  [http://localhost:3000](http://localhost:3000)
